# Sistema de Monitoreo de Calidad del Aire con ESP32

Este proyecto consiste en el diseño e implementación de un sistema embebido para el monitoreo de la calidad del aire utilizando un microcontrolador ESP32. La solución integra sensores ambientales que permiten medir variables como concentración de partículas, presencia de gases y condiciones ambientales, con el objetivo de calcular un índice representativo del estado del aire en tiempo real.

El sistema fue desarrollado bajo una arquitectura modular que separa la adquisición de datos, el procesamiento de la información y la generación de alertas. Según el nivel detectado, el prototipo activa indicadores visuales en una pantalla LCD, además de alertas sonoras mediante un buzzer.
